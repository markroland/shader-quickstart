# Shader Quickstart using Three.js

This project uses [Three.js](https://threejs.org) [Vite](https://vitejs.dev) to compile and serve
a OpenGL Shading Language (GLSL) shader using WebGL.

## Install

```
npm install
```

## Run

```
npm run dev
```
